# STYLES

LINEスタンプ画像生成のスタイル定義。

## コアスタイル（5種）

| スタイルID | 名前 | 頭身 | 説明 |
|-----------|------|------|------|
| `sd_25` | **標準ちびキャラ（推奨）** | 2-2.5 | バランス良く汎用性高い |
| `sd_10` | **超デフォルメ** | 1-1.5 | 感情パーツ最大、表情ドン系 |
| `sd_30` | **ジェスチャー重視** | 3-4 | 衣装やジェスチャーを表現しやすい |
| `face_only` | **顔だけ（表情ドン）** | 顔のみ | 表情で勝負、文字なしに最適 |
| `yuru_line` | **ゆる線画** | 2-3 | 手描き風のゆるいタッチ。10代に人気 |

## Advancedスタイル（3種）

| スタイルID | 名前 | 説明 | 注意事項 |
|-----------|------|------|---------|
| `semi_50` | **衣装重視（5頭身）** | 4.5-5頭身。衣装詳細に | シンプル寄せ必須 |
| `pixel_art` | **ピクセルアート** | ドット絵、レトロゲーム風 | 本人性保持困難 |
| `illustration_rich` | **イラストリッチ** | 華やか・ガチャ絵風 | 失敗率高め |

## 旧スタイルIDの互換性（自動変換）

| 旧ID | 新ID |
|------|------|
| `ultra_sd`, `extreme_chibi`, `choi_sd` | → `sd_10` |
| `standard_sd`, `puni`, `ball_joint` | → `sd_25` |
| `tall_sd`, `mini_chara` | → `sd_30` |
| `semi_deformed` | → `semi_50` |
| `gacha` | → `illustration_rich` |

## モディファイア

### テキストモード（text_mode）

| 値 | 名前 | 説明 |
|----|------|------|
| `deka` | でか文字（デフォルト） | 40%以上の大きな太文字、縁取り付き。タブサイズでも読める |
| `small` | 通常 | 小さめの手書きテキスト |
| `none` | なし | テキストなし（face_only向け） |

### アウトライン（outline）

| 値 | 名前 | 説明 |
|----|------|------|
| `bold` | 太フチ（デフォルト） | 5-8pxの太い白フチ。どの背景でも見やすい |
| `white` | 白フチ | 2-3pxの標準フチ |
| `none` | なし | フチなし |

## CLI オプション

```bash
python generate_stamp.py --style <style_id> --text-mode <mode> --outline <outline>
```

例:
```bash
python generate_stamp.py --package input/photo.jpg --style yuru_line --text-mode deka --outline bold
```

## スタイル選択フロー

1. **参照画像のテイストを判定**
   - ドット絵/ピクセルアート → `pixel_art`
   - 写真/リアル調 → `sd_25` または他のちびキャラスタイル
   - 手描き/ゆるい線画 → `yuru_line`
   - イラスト/アニメ調 → 参照画像のテイストに近いスタイルを選択

2. **ユーザー確認**
   - 「このままのテイストで」→ 参照画像に合わせて自動選択
   - 「ドットで」→ `pixel_art`
   - 明示的にスタイルIDを指定 → そのまま使用
